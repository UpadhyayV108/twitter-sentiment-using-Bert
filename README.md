#  Twitter Sentiment Analysis using BERT

This project uses BERT and HuggingFace Transformers to classify tweets as **Positive**, **Negative**, or **Neutral**. Built for reproducibility and public impact, it includes a Colab notebook, PDF report, and setup instructions.

##  Project Structure
- `notebook.ipynb` — Colab notebook with full pipeline
- `requirements.txt` — Python dependencies
- `report/` — PDF export for campus submission
- `README.md` — Project overview and instructions

##  Features
- Preprocessed Twitter dataset
- BERT fine-tuning with HuggingFace
- Sentiment classification
- Accuracy and loss visualization
- Reproducible workflow via Colab
